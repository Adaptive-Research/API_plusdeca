Utilisateurs

Le module UTILISATEURS comporte 3 tables dont : utilisateur_compte, utilisateur_session ,utilisateur_infos.

La table utilisateur_compte contient le paramétrage des utilisateurs à propos de leur email, leur langue parlée et leur recommandation.

La table utilisateur_session permet le stockage en session du token d'un utlisateur

La table utilisateur_infos permet le stockage en base de toutes les info rmations d'un utlisateur.
